Description

MultiWindow
Multi Window Is An App By Which You Can Create Shortcuts Of Your Installed Apps Icon Into Mutiple Window Tray. And By Clicking On These Shortcuts, You Can Easily Switch From One Task To Another During Your Current Task.
Extra Features ::
# No root required
# You Can Increase And Decrease Speed Of Multi Window Tray.
# You Can Arrange Icons.
Types: Most used app, Recently used app and Alphabetical Order.
# You Can Select Many Different Default Themes.
# You can hide and unhide On long pressing multi window Sidebar icon. (Follow Help )
# Long press on slide bar item and move on Scree to open app.
# You can change opacity of slide bar.
# Customize color, animation speed,order and opacity.
# Service start automatically when phone restart Fix.
Note : This MultiWindow app does't split your screen in two parts. This is hardware dependencies.
