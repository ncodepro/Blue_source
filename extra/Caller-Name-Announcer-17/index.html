Description

Caller Name Announcer is highly preferential in situations like if you are driving and phone is in pocket or some one sends you an SMS while your phone is in other room. Caller Name Announcer simply speaks it loud for you.
Caller Name Announcer speaks every thing while some one is calling you or send you a message, you will identify it with out looking to your smart phone. Caller Name Announcer uses the built-in Android text-to-speech engine to announce the incoming calling person name or SMS sender name and contents of the received SMS. Caller Name Announcer will not work if your phone does not have text-to-speech library but this is not a big problem, It can be easily downloaded from Google Play Store.
Caller Name Announcer Speaks,
- Name of calling person if exist in contacts
- If does not exist in contacts, calls that " An unknown number "
- Speaks received SMS sender name and contents
Features:
Easily customize your Caller Name Announcer application
- Enable / disable speaking caller name
- Enable only speaking SMS sender name
- Enable / disable announcement of received SMS and its contents
- Selecting Different Ring tones and more Ring tone Settings
- Volume Control Settings
- Text-to-Speech Settings
